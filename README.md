# 📘 Tarefas de Linguagem de Programação(LPR)

Este repositório contém as tarefas desenvolvidas durante o curso de Desenvolvimento de Sistemas, da matéria Linguagem de Programação, organizadas por trimestre.

## 📂 Estrutura do Repositório

- `1o_Trimestre/` — Exercícios realizados no primeiro trimestre.
- `2o_Trimestre/` — Exercícios realizados no segundo trimestre.

> ⚠️ Outras pastas e arquivos foram intencionalmente ocultados via `.gitignore` para manter o foco nas tarefas principais.

## 🛠️ Tecnologias Utilizadas

- **Linguagens:** C# e C++
- **IDE:** Visual Studio
- **Controle de Versão:** Git
